RustBeds branding assets live in the main repository under `docs/branding`.

## Recommended assets

| Asset | Size | Suggested use |
| --- | ---: | --- |
| `rustbeds-banner.png` | 855 x 162 | Wiki home page, repository social previews, wide page headers. |
| `rustbeds-logo-primary.png` | 938 x 200 | Primary documentation logo when horizontal space is available. |
| `rustbeds-logo-alt.png` | 512 x 107 | Smaller horizontal logo placements. |
| `rustbeds-logo-stacked.png` | 162 x 270 | Narrow layouts, sidebars, profile-style docs. |
| `rustbeds-minecraft-logo.png` | 564 x 89 | Minecraft-focused listings or short headers. |
| `rustbeds-icon.png` | 145 x 143 | Avatar-like uses and compact identity marks. |
| `rustbeds-badge.png` | 184 x 205 | Badge-style promo or plugin listing panels. |
| `rustbeds-discord-banner.png` | 855 x 101 | Discord/server community banner. |
| `server-icon.png` | 64 x 64 | Minecraft server icon use. |

## Usage notes

- Use the banner or primary logo for first-page recognition.
- Use the icon only where space is tight.
- Prefer the raw GitHub URLs in wiki pages.
- Keep screenshots separate from branding assets if gameplay screenshots are added later.
